Cores semânticas — Status forense
status-ok
Evidência confirmada, vestígio íntegro, hash válido. Oliva escura — não verde festivo.
status-warn
Atenção. Metadado ausente, data inferida, anexo parcial. Âmbar envelhecido.
status-alert
Crítico. Quebra de cadeia de custódia, hash divergente, fonte não autenticada.
status-muted
Neutro. Em processamento, aguardando revisão, fora de escopo.
